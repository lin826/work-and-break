Work and Break
增進社群溝通以減緩拖延焦慮之系統設計與實作

摘要
  隨著網路科技與行動裝置的普及，現代文明病「拖延症」也跟著普遍發生，其中，來自社群網路所提供的內容與社交活動是造成拖延的一大因素。為了降低社群網路造成的拖延所衍生的負面作用，如何與適度與社群互動、做好自我時間管理，成了重要議題。目前 時間管理方法千奇百樣，若能結合時間管理與電腦輔助溝通(CMC)，將可以為透過訊息的 揭露讓溝通更加的順暢，以減少來自社群網路的誘惑造成的工作中斷與效率低落。
  此研究結合心理學理論、社群平台運用及人機互動機制，以跨領域科技整合的方式，試圖使用鍵盤與滑鼠監測來取得人的焦慮狀態，並將焦慮資訊適度揭露以幫助社群了解使用者的當前工作情形，藉由分享平台促進網路使用者之間的溝通效果，而系統的記錄能協助使用者自我覺察，排出更適合自己的計劃。期望以人機互動的角度，並用時間管理術帶來科技化的優點。

	  關鍵字:拖延症，社群平台，焦慮，時間管理
